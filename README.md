# 🎲 Number Guessing Game

A colorful and emoji-filled number guessing game🎀

## ✨Features

- Random numbers between 1-10
- Get hints if you're too high or too low
- Celebrate your win 🎉
- 💻Deployed using Render

- Flask 🌐
- HTML (via index.html)
